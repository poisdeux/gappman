Panel should support dynamic loading of modules that can be added to the panel area.

Basic functions:

gm_module_init()
Initializes the module. As a minimal action it should create the widget.

gm_module_start()

gm_module_stop()

gm_module_set_icon_size()

//Following function should return the widget that should be added to the panel.
gm_module_get_widget()


Adding an applet should be done in the main gappman config file using the panel sub-element:

  <panel width=100% height=20%>
    <applet>
      <name>Network manager</name>
      <module>./widgets/netman/gm_netman.so</module>
    <applet>
  </panel>
